Local preview
1. Open a terminal in this folder.
2. Run a simple HTTP server (Python 3):
   - Windows: `python -m http.server 8000`
   - macOS/Linux: `python3 -m http.server 8000`
3. Open http://127.0.0.1:8000 in your browser.
