Lyra Starter Game

The Lyra Sample Project, developed by Epic Games, is an extensive project used to showcase many of the features of Unreal Engine. Many of the modular features that Epic uses to build it's games (such as Fortnite) are used to give an example of the optimal development workflow in Unreal Engine for modern game development.
